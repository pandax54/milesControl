## Recommendations

2. **Document bot setup**: Consider adding a comment block above the webhook handler with instructions for setting up the Telegram webhook with the bot token, e.g.:
   ```
   // Setup webhook via: curl -X POST https://api.telegram.org/bot<TOKEN>/setWebhook -d url=https://your-domain/api/webhook/telegram -d secret_token=<CRON_SECRET>
   ```

3. **Monitor retry exhaustion**: The retry logic logs warnings but doesn't increment a metric. Consider adding Posthog/observability event when retries are exhausted for production alerting.
